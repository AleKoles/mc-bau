# MC Bauchemie Design System

Storybook component library for [MC-Bauchemie](https://www.mc-bauchemie.com/) — one of the world's leading manufacturers of construction chemical products and technologies.

## What's Inside

- 🎨 **Colors** — Primary, secondary, and system color palette
- 🔤 **Typography** — Desktop and mobile type scales
- 🧩 **Components** — UI components *(work in progress)*

## Built With

- [Storybook](https://storybook.js.org/)
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- Design tokens sourced from Figma via [Figma MCP](https://www.figma.com/blog/introducing-figma-mcp-server/)

## Getting Started

```bash
npm install
npm run storybook
```

Storybook runs at `http://localhost:6006`

## Build

```bash
npm run build-storybook
```

Output goes to `storybook-static/`

## Live

Deployed on Vercel — [mc-bau.vercel.app](https://mc-bau.vercel.app)
